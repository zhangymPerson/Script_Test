# 多种 sh 的对比

- csh

	sh 等价于 bash --posix
	/bin/sh 相当于 /bin/bash --posix，所以二者的一个区别就是有没有开启 POSIX 标准模式。
	
	“sh”并不是一个程序，而是一种标准（POSIX），

	这种标准，在一定程度上（具体区别见下面的“Things bash has that sh does not”）保证了脚本的跨系统性（跨UNIX系统）
